# GwaIO Architecture Test

This repository is a proof of concept for the re-architecture of GwaIO.
The goal is to redesign the framework using a hexagonal architecture, allowing the core application to remain independent from external technologies and integrations.
The system is intended to be fully addon-driven, enabling features, DCC integrations (Maya, Blender, etc.), project management systems, and custom studio workflows to be added or replaced without modifying the core codebase.
This repository is used for experimentation, validation, and development of the next-generation GwaIO architecture

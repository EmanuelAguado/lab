# Pipeline Node System Refactor Test

This repository contains a refactoring test of the original node-based pipeline configuration system.

## Shortcuts

| Shortcut | Action |
|-----------|----------|
| `Ctrl + L` | Load configuration |
| `Ctrl + S` | Save configuration |
| `Ctrl + H` | Print history |
| `Ctrl + C` | Copy selected nodes |
| `Ctrl + V` | Paste nodes |
| `Ctrl + X` | Cut selected nodes |
| `Space` | Pan / move the graph view |
